# NeurSC 训练文档 4 —— Exp-6c 快速探路（50 轮）

**目的**：v1.0.2 简化架构（`--num_inter_layers 4 --norm_type none`）下先跑 50 轮看曲线趋势，
判断加深 + 残差是否有收益，再决定是否上 120 轮完整对照。

**基线对照**：Exp-5（v0.15，2 层裸堆叠，120 轮）`test_rel_err = 0.119, best_ep = 115`。

---

## 训练命令

```bash
nohup python -u main.py \
  --off_file dataset/EP_low/EP_low.off --file_folder . \
  --max_depth 3 --capacity 512 \
  --distance_samples 50000 \
  --learning_rate 0.001 \
  --lr_scheduler plateau --lr_patience 5 --lr_factor 0.5 --min_lr 1e-6 \
  --dropout_ratio 0.1 \
  --num_epoch 50 \
  --batch_size 16 \
  --num_inner_layers 2 --num_inter_layers 4 \
  --norm_type none \
  --early_stop_patience 15 \
  --highway_k 3 --inner_mode partition --loss_type log_l1 \
  --device cuda \
  > logs/EP_low_exp6c_v102_inter4_none_50epoch.log 2>&1 &
```

**预估时间**：约 10~11 小时（每轮 ~750s × 50 轮，命中缓存跳过预处理）。
