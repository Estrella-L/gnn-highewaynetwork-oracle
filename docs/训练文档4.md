# NeurSC 训练文档 4 —— Exp-7 快速探路（v1.0.3，50 轮）

**目的**：v1.0.3 回滚到 v0.15 极简结构 + **InterGNN 单加 1 层**（`--num_inter_layers 3`），
其它所有超参与 Exp-5 严格一致。**唯一变量是 InterGNN 深度**。50 轮看曲线，判断"加深 1 层"是否有增益。

**基线对照**：Exp-5（v0.15，2 层）`test_rel_err = 0.119, best_ep = 115, 120 轮`。

**判据（重要）**：
- ✅ epoch 25-35 突破 val_mae 2500 平台，加速下降到 1500 以下 → 加深方向有效
- ❌ epoch 30+ 仍卡 2500 附近 → InterGNN 加深无益，接受 v0.15 深度是甜点区

---

## 训练命令

```bash
nohup python -u main.py \
  --off_file dataset/EP_low/EP_low.off --file_folder . \
  --max_depth 3 --capacity 512 \
  --distance_samples 50000 \
  --learning_rate 0.001 \
  --lr_scheduler plateau --lr_patience 5 --lr_factor 0.5 --min_lr 1e-6 \
  --dropout_ratio 0.1 \
  --num_epoch 50 \
  --batch_size 16 \
  --num_inner_layers 2 --num_inter_layers 3 \
  --early_stop_patience 15 \
  --highway_k 3 --inner_mode partition --loss_type log_l1 \
  --device cuda \
  > logs/EP_low_exp7_v103_inter3_50epoch.log 2>&1 &
```

**预估时间**：约 10~11 小时（每轮 ~750s × 50 轮，命中缓存跳过预处理）。3 层比 2 层慢约 15%，
比 4 层快约 25%。

**注意**：拉最新代码后旧 checkpoint 全部作废（v1.0.x 的 state_dict 结构变了）。从头训。
